What we do

We help companies building Data Platform for Machine Learning Features (Feature Store) with following features:

  • Combining features from Batch and Stream Data
  • Scalable access to feature data which serving and training
  • Time travel to build accurate training data sets
  • Monitoring Feature Skew to detect Data Drift and breakages
  • Sharing, discovery and re-use for Feature Data
  • Feature store that integrates with common Data Infrastructure and ML platform like Kubeflow, Databricks and AWS Sagemake

What we do

Our mission is to help companies to build Data Platform for Machine Learning Features (Feature Store) that supports:

  • Combining features from Batch and Stream Data
  • Scalable access to feature data while model serving and model training
  • Time travel to build accurate training datasets
  • Monitoring Feature Skew to detect data drift and breakages
  • Sharing, discovery and re-use for feature data across org
  • Feature store that integrates with common data infrastructure and ML platforms like Kubeflow, Databricks and AWS Sagemaker

About Us
